Phronex

From Greek φρόνησις ('Phronesis')

Practical wisdom in decision-making, planning, and budgeting

Key features

  • Project and portfolio management.
  • AI-powered budget planning.
  • Expense tracking and categorisation.
  • Resource planning.
  • Clean, responsive and accessible UI.
  • Cloud ready.

Phronex highlights

Projects overview - with coloring adjust to budget adjustment.

User experience

The UI allows free text search, date filtering and sorting of projects, resources, and expenses across all of its screens.

AI features

Based in the declared scope and time duration, the application can propose an initial budget thanks to AI*.




 

 

 

 




*Note this feature is in beta

Consistency

Additionally it allows to manage a single workforce* that can be distributed across projects with generic but configurable roles.

* Limited features on v1.0

Phronex allows a consistent portfolio management thanks to using a single set of reusable cost categories across the different projects.

Accessibility

The UX/UI considers a diverse set of users with different needs. All reusable components adhere to WAI-ARIA design principles thanks to ShadCN/UI reusable components.
 

 


It also has a high-contrast dark mode to protect user vision.

Technical features

Technology stack

The application has considered a varied but consistent stack, based on full-stack javascript / typescript.

Solution architecture - TEST

The product is deployed in AWS (using EC2).

It is running fully dockerized.

It also interacts with OpenAI as an external service via API keys.

Solution architecture - PROD

Ideally this application should serverless. A solution based on OpenNext (for pushing server actions into AWS ambda has been looked into).

 

Navigation

Taking advantage of NextJS 15 as a well structured and opinionanted React framework, navigation has been designed to reuse layout elements across the screens.
